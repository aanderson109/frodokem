## Project Structure

- `optimized/` contains the future low-level implementation of FrodoKEM that is optimized for specific goals that will be defined at a later date. This will be written in C.
- `reference/` contains the "gold standard model" of FrodoKEM as defined in the third-round specification submitted to NIST during the PQC standardization process. It is written in Python.
